# Netflix Clone (Frontend)

A frontend-only Netflix homepage UI clone built using HTML and CSS.  
This project focuses on recreating the visual layout and styling of the Netflix landing page.

---

## Project Overview

This project was developed to practice frontend web development skills, with emphasis on
layout design, responsive styling, and clean user interface implementation.
It does not include backend functionality or JavaScript logic.

---

## Technologies Used

- HTML5
- CSS3

---

## Features

- Netflix-inspired user interface
- Responsive layout design
- Clean and structured HTML
- Modern CSS styling techniques
